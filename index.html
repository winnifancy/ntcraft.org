01 — 關於本協會

關於我們

社團法人南投縣手工藝發展協會致力於推廣地方工藝文化，結合文化教育、人才培育、社區服務與公益資源，透過課程、展覽、體驗活動及跨領域合作，推動手工藝的傳承與創新。

我們相信，工藝不僅是技藝的延續，更承載著生活美學、文化記憶與社區情感，因此長期深耕在地，陪伴居民、親子、樂齡族群及教育工作者共同學習與成長，讓手工藝融入生活，成為促進文化傳承、世代交流與地方永續發展的重要力量。

我們期望持續整合各界資源，建立完善的手工藝培育與推廣機制，打造南投縣具專業、公信力與影響力的手工藝培訓與推廣平台。

願景

成為南投縣手工藝人才培育、文化傳承與公益教育的重要推動者，讓工藝成為連結人、土地與文化的永續力量。

我們的宗旨與任務

MISSION 01

推廣手工藝學習

辦理有關手工藝之研習、觀摩、講座、聯誼及解說訓練等活動，以提升民眾對手工藝的興趣和瞭解。

MISSION 02

促進生活休閒

辦理促進地方民眾生活休閒等活動，豐富日常生活的美感與樂趣。

MISSION 03

社教與技能訓練

配合政府政策舉辦各項社教研習、專業技能訓練，以增進生活內涵。

MISSION 04

公益推廣活動

接受委託辦理有關手工藝推廣等公益活動，將工藝資源回饋社會。

MISSION 05

社區與家庭關懷

重視社區人文與教育訓練服務，關心幼兒與兒童、家庭或教保服務人員福祉相關事項。

MISSION 06

其他宗旨事項

其他合於本會宗旨之事項，持續因應地方需求拓展服務範疇。

02 — 推廣項目

從課程到公益，工藝落地生活的方式

圍繞宗旨任務，協會規劃多元推廣項目，串連學習、休閒、社教與公益，陪伴不同年齡層走進手工藝的世界。

PROGRAM / 01

手工藝研習課程

毛線編織、染布、金屬工藝、複合媒材...等基礎與進階課程，培養在地工藝人才。

PROGRAM / 02

觀摩交流講座

邀請工藝職人與在地創作者分享經驗，促進跨領域對話與觀摩。

PROGRAM / 03

社區公益服務

受託辦理工藝推廣公益活動，將技藝帶入社區角落。

PROGRAM / 04

親子家庭工作坊

關懷幼兒、兒童與家庭，設計適合親子共學的手作體驗。

PROGRAM / 05

教保人員培力

協助教保服務人員增能研習，將工藝融入教學現場。

PROGRAM / 06

專業技能訓練

配合政府社教政策辦理專業技能訓練，銜接生活與職能發展。
